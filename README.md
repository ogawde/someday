# Someday

A museum of unfinished things — preserve abandoned projects as exhibits, browse by wing, and request to continue the work via in-app chat.

## Setup

```bash
cp .env.example .env.local
npm install
npm run db:push
npm run dev
```

See [docs/context.md](docs/context.md) for architecture, env vars, and scripts. Product brief and hackathon rules live in `docs/`.
